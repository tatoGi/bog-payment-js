# Changelog

All notable changes to this package are documented in this file.

## [1.0.0] - 2026-03-05

### Added

- Initial npm release for BOG iPay JavaScript client.
- OAuth2 client credentials authentication.
- Order create and payment details API methods.
- Saved-card payment operations and preauthorization methods.
- Built-in timeout, token cache, and structured error handling.
- Node-based test suite.
